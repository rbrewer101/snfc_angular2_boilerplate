Interface TypeDecorator

An interface implemented by all Angular type decorators, which allows them to be used as ES7 decorators as well as Angular DSL syntax.

DSL syntax:

var MyClass = ng
  .Component({...})
  .View({...})
  .Class({...});

ES7 syntax:

```

ng

.Component({...})

ng

.View({...}) class MyClass {...} ```
